Introduction to HTML

HTML (HyperText Markup Language) is the standard language for creating webpages.

Headings

HTML provides six levels of headings, from <h1> to <h6>.

<h1>This is Heading 1</h1>
<h2>This is Heading 2</h2>

Text Formatting

Use formatting tags like <b> for bold and <i> for italics.

<p>This is <b>bold</b> and <i>italic</i> text.</p>

Links

Use the <a> tag to create hyperlinks.

<a href="https://example.com">Visit Example</a>

Images

Use the <img> tag to display images.

<img src="image.jpg" alt="Example Image">

Tables

Create tables using <table>, <tr>, and <td> tags.

<table border="1">
  <tr>
    <td>Row 1, Cell 1</td>
    <td>Row 1, Cell 2</td>
  </tr>
</table>
